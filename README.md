# Watch Me
## Second challenge of the track of [ReactJS](https://pt-br.reactjs.org/) by Ignite / [Rocketseat](https://github.com/Rocketseat). The project's purpose is to refactor the application by dividing it into at least two main parts: the sidebar and the main content that has the header and movie listing.
